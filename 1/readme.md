---

## 변수,상수

> - 변수는 **변할수 있는 수**, 상수는 **변하지 않는 수**

변수:

```
x,y,...
```

상수:

```
1,5,8,2,13....
```

---

## 대입연산자

> - = ( **??를 ??에 대입한다** )

예시:

```
x = 3(x에 3을 대입), x = 15(x에 15를 대입)
오른쪽을 왼쪽에 대입
```

## 입력

> - a = intput() 입력은 **함수**이며 **표준 출력 공간을 읽는다**
>   예시:

```
a = input()
a에 함수(입력)을 대입
```
